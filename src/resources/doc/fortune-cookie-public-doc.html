Fortune Cookie for Jira

About the app

Add a spark of inspiration and fun to your Jira projects with the Fortune Cookie app! Harness OpenAI to generate project-specific prophecies based on lightweight, non-sensitive metrics (e.g. issue counts). Whether you're refining your backlog or managing your board, open a fortune cookie for a unique blend of motivation and insight.

We do not store or process any sensitive project information! We only process the following data with OpenAPI chat completion:

  • Project key
  • Project type (e.g. ‘software’, ‘business’, …)
  • Issue type names (e.g. ‘task’, ‘bug’, ‘story’,…)
  • Issue type descriptions
  • Issue type counts by status categories (i.e. ‘to do’, ‘in progress’, and ‘done’)
